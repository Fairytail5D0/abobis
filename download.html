Two-way sync

Changes on either side are instantly mirrored. No manual steps, no conflicts.

Background mode

Runs silently in the system tray. Zero performance impact on your workflow.

Exclude filters

Skip temp files, logs, and system folders with simple glob patterns.

Scheduled sync

Set hourly, daily, or on-change triggers. Works with Windows Task Scheduler.

No cloud required

Everything stays local. LAN, USB drives, mapped network paths all supported.

Low memory footprint

Under 12 MB RAM at idle. Written in native C++ for maximum efficiency.
